# RusselsParadoxVisualiser
A small simple visualization of Russels Paradox made for easy understanding. :)

Bertrand Russel (1872-1970) was a profound British mathematician and philosopher highly known in the mathematical and philosophical world for his work in advancing set theory. He determined the fundamental rules concerning the requirements of establishing a set. 

Start with defining a set, a set is a collection of objects, for example:
A set X is simply a collection of things, these things are referred to as the elements of set X. To describe a set formally we list it with curly braces surrounding the list as so:
{a, b, c} This is describing the set with the elements a, b and c. Anything can be a set if defined, like the collection of all prime numbers is a set {2, 3, 5, 7, 11, 13, . . .}. Now, when talking about how anything can be a set, say I have three different coloured pens. That is a valid set. However, sets are not restricted to only objects or only numbers. If willing, the set: {1, apple, carrot} is a valid set. We can also have the set of all imaginary objects which may be an infinite number of imaginary elements. 

Formal set notation is a way of expressing sets that can not be simply written out. Instead, it is written as so: {x : x is an even number}. This is read as the set of all x’s, such that (the colon expresses “such that”) x is an even number. We use the expression “contains” to describe the objects in the set, the set contains all even numbers. Noticeably, sets are something we come to deal with frequently like, the set of all students in a defined public school is large. We are talking about the group that is large. The grounding of basic set theory has been established to formally introduce the rules which concern the Paradox.
